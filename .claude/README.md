# `.claude/`

Team-shared agent configuration: `settings.json`, plus optional `skills/` and `commands/`. Personal, machine-local settings go in `settings.local.json` (gitignored). This dir holds *decisions* shared with the team, so it is committed — unlike caches and `.env`.
